The source codes of this project are placed in this directory. Source code directory stores the latest **complete** firmware of whole project. 

- This project is built around Arduino Uno or Pro mini.

- The rest codes are also complete and will work but are outdated and have bugs. The code in sourcecode sirectory stores updated, bug free, complete code; aka, the firmware of the project. The firmware can be uploaded to pro mini with the help of a USB to TTL convertor.
